### Trends in COVID-19 variants and prediction of next wave

This is a project on the analysis of COVID-19 variants and prediction of the next wave. The project is divided into two parts:
1. Analysis of COVID-19 variants using data analysis and visualization techniques.
2. Prediction of the next wave of COVID-19 using simulations in the SEIR model.

The project is a part of the course "Disaster Management" at IIIT Hyderabad.
